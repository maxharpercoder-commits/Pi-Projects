# Pi-Projects

---

Welcome to Pi-Projects! this is where I share my discoveries and code that I use, I use a Raspberry Pi 5 (64-bit) so not all of this will work, but most of it should.

---

### Install:
`wget -qO- https://raw.githubusercontent.com/Botspot/pi-apps/master/install | bash`
##### Under the hood:
`git clone https://github.com/Botspot/pi-apps && ~/pi-apps/install`
### Uninstall:
`~/pi-apps/uninstall`
for more information see https://pi-apps.io/install/
